# aronica

A theme for Hugo.

## Features

* 简约设计(Less is more)
    - 或许要换一种方式来
    - 干净的页面, 尤其是博文内容上, 专注于内容, 禁止多余的元素, 甚至可以缺少一些功能(比如导航, 用浏览器自带的就好)
* 支持黑暗模式
* 响应式设计(支持电脑, 平板, 手机)
* 支持代码主题


## Installation

```bash
git submodule add https://github.com/arong/hugo-theme-aronica.git
```
