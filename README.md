# hugo 主题 Aronica

**DO NOT USE THIS THEME. It's under development.**

**不要使用此主题, 正在开发中.**

## 样例网站

[https://www.arong-xu.com](https://www.arong-xu.com/)

## Features

- 支持黑暗模式
- 响应式设计(支持电脑, 平板, 手机)
- 支持代码主题

## Installation

```bash
git submodule add https://github.com/arong/hugo-theme-aronica.git
```
