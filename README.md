## Features

- 支持黑暗模式
- 响应式设计(支持电脑, 平板, 手机)
- 支持代码主题

## ShortCodes

1. 多标签代码
2. 引入公共文件, 比如专栏目录, 在专栏中的每一个文件中都引入相同的目录
3. `PostRef`自动引用帖子的 title 和链接

## 响应式布局

目前采用的是 DaisyUI 的 drawer 布局做的

### header 元素

1. 公共部分:
1. Search bar
1. 黑白切换按钮
1. 菜单栏
1. 对于专栏, 存在一个专栏的目录, 在移动端上面显示出一个按钮, 点击就会弹出 drawer
